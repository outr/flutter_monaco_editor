# Security Policy

## Supported Versions

Only the latest published `0.x` release on pub.dev is supported while the
package is pre-1.0. Fixes land on `main`; published packages are
superseded by newer ones rather than patched in place.

| Version | Supported |
| ------- | --------- |
| 0.5.x   | ✅        |
| < 0.5   | ❌        |

## Reporting a Vulnerability

Please report suspected vulnerabilities **privately** — do not open a
public GitHub issue for anything exploitable.

1. Preferred: open a [GitHub Security Advisory](https://github.com/outr/flutter_monaco_editor/security/advisories/new)
   on this repository.
2. Alternatively: email `security@outr.com` with subject
   `flutter_monaco_editor: <short description>`.

Include:

- Affected version(s)
- A minimal reproduction or proof-of-concept
- The impact you observed

We aim to acknowledge within **5 business days** and publish a
coordinated fix + advisory within **30 days** of a verified report.
